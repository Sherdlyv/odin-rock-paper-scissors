# odin-rock-paper-scissors

A classic Rock, Paper, Scissors game played directly in the browser's console. This project was built as part of **The Odin Project** foundations curriculum.

## Features

- **5-Round Gameplay**: The game automatically loops through 5 rounds.
- **AI Choice**: The computer choice is randomly generated.
- **Input Validation**: Human inputs are case-insensitive (e.g., "Rock", "rock", and "ROCK" all work).
- **Score Tracking**: Scores are updated in real-time after each round, with a final winner announced at the end.

##  Technologies Used

- **JavaScript (ES6+)**: Game logic, functions, loops, and conditionals.
- **HTML5**: Basic structure to link and load the script.
- **Git & GitHub**: Version control.

##  Key Takeaways

- Learned how to generate random outcomes using `Math.random()`.
- Mastered function nesting and variable scope.
- Implemented `for` loops to control game flow and state.

## How to Play

1. Open the `index.html` file in your web browser.
2. Open the developer console (`F12` or `Right-Click -> Inspect -> Console`).
3. Follow the pop-up `prompt()` instructions to enter your choices!
